# Team Block 
The Team Block is an element designed to showcase your team members. You can add as many team members as you'd like, with each member featuring a profile picture, name, and title.
![Team Block](/11-team-block/team-block.jpg)

All blocks, including Team Block, can be visually rearranged using a simple drag-and-drop interface. This allows for easy organization and customization of your content layout.

## Block Options

Each block comes with a set of features accessible through the three-dot icon located in the top right corner. This icon is visible when the block is in Edit mode, among other scenarios.
